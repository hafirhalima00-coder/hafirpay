API Integration

Integrate HafirPay into your store.

Your Merchant API Key

This is your merchant key. Use it for direct integrations. For AI agents, create Agent Keys in the Agent Keys tab.

Create a Payment

curl -X POST https://hafirpay.app/api/transactions/create \
  -H "x-api-key: hp_your_merchant_key" \
  -H "Content-Type: application/json" \
  -d '{
    "amount": 25.00,
    "currency": "USD",
    "description": "Order #1234",
    "customerName": "John Doe",
    "customerEmail": "john@example.com"
  }'

Send Invoice via Email

curl -X POST https://hafirpay.app/api/transactions/HP-xxx/send-invoice \
  -H "x-api-key: hp_your_merchant_key"

Sends a professional HTML invoice email to the customer's email on file.

Agent API Usage

# AI Agent creates a payment link
curl -X POST https://hafirpay.app/api/agent/transactions/create \
  -H "x-agent-key: hak_agent_key_here" \
  -H "Content-Type: application/json" \
  -d '{
    "amount": 50.00,
    "currency": "USD",
    "description": "Consulting session",
    "customerEmail": "client@company.com"
  }'

# Agent sends invoice automatically
curl -X POST https://hafirpay.app/api/agent/transactions/HP-xxx/send-invoice \
  -H "x-agent-key: hak_agent_key_here"

How Invoicing Works

  1. Configure your SMTP settings above
  2. Create a payment with the customer's email address
  3. Click "Send Invoice" on the transaction or use the API
  4. Customer receives a professional HTML invoice with a Pay Now button
  5. They click the link, see bank details, and transfer directly to you
